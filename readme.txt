== Description ==

Easily delete large swaths of tags with all sorts of options, like auto delete batches and WordPress table optimization.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Add the shortcode, with optional settings, to a page and open the page.
